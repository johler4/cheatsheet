What is CSS?

CSS stands for cascading style sheets and is the language that we use to style a Web page. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS is able to control the layout of multiple web pages all at once by them all being linked to the same external stylsheet.

CSS Syntax

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

Example CSS declaration block:

.section {
color: blue;
background-color: #f075d7;
align-text: center;
font-weight: 700;
}

CSS Pseudo-classes

A pseudo-class is used to define a special state of an element.

For example, it can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

The syntax of a psuedo-class:

selector:pseudo-class {
property: value;
}

Anchor pseduo-classes

Links can be displayed in diferent ways:

/*unvisited link */
a:link {
color: #ff0000;
}
/*visited link*/
a:visited {
color: #00ff00;
}
/*mouse over link*/
a:hover {
color: #ff00ff;
}
/*selected link*/
a:active {
color: #0000ff;
}

the :first-child pseudo-class

The :first-child pseudo-class matches a specified element that is the first child of another element.

In the following example, the selector matches any < p > element that is the first child of any element:

p:first-child {
color: blue;
}

In this example, the selector matches the first < i > element in all < p > elements:

p i:first-child {
color: blue;
}

This example is where the selector matches all < i > elements in < p > elements that are the first child of another element:

p:first-child i {
color: blue;
}
